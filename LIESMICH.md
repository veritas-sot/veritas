# Installation

## Python Environment

Um die Library zu installieren setzt man am besten ein lokales Python-Environment ein. So kann man zum Beispiel (mini)conda nutzen.

Mit

conda create --name veritas python=3.11

wird ein neues Environment mit dem Namen 'veritas' und der Python Version 3.11 angelegt.

## Installation der Library

Mit 

poetry install

wird die Library installiert.
